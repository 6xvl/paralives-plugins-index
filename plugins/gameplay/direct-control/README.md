# Direct Control (by Rockz)

WASD direct character movement for Paralives — third-person camera, in-game settings
menu with rebindable keys, sprinting, walk styles, object-fading toggle, auto-doors,
improved collision.

**Author:** Rockz · **Source/updates:** https://github.com/RockZ-01/DirectControl

Featured in the 6ix Plugin Hub as a community collaboration. Mod is authored and
maintained by Rockz; hosted here for one-click install.
